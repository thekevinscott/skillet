---
name: docs-update
description: Update README and spec docs after a feature ships.
---

1) Scan the feature spec for user-facing changes.
2) Update `README.md` (quickstart, usage examples, env vars/config).
3) Update/add pages under `spec/`.
4) Add an entry to `spec/changelog.md`.
5) Record updates in the feature spec Docs/README section.
