# PDF Extraction

Standard: `python "$CLAUDE_PLUGIN_DIR/scripts/pdf_extract.py" "file.pdf"`
Unified: `python "$CLAUDE_PLUGIN_DIR/scripts/pdf_extract_unified.py" "file.pdf"`

Read the extracted .txt or _unified.md, not the PDF.
