# Reminders Skill

Access Apple Reminders via the `reminders` CLI. Run `reminders --help` for all options.

## Output Format

Always present reminders in a markdown table with recurring indicator:

| Reminder | Due | Recurring |
|----------|-----|-----------|
| Task name | today 3:00 PM | |
| Weekly standup | tomorrow | ↻ |
| Overdue item | 2 days ago | |
