# Skill: bugfix-with-repro

## Proposito

Corrigir bug garantindo que nao volte (teste de regressao).

## Fluxo

1. WRITE REPRO TEST -> Escrever teste que reproduz o bug
2. VERIFY BUG EXISTS -> Confirmar que teste falha
3. FIX BUG -> Corrigir o codigo
4. VERIFY BUG FIXED -> Confirmar que teste passa
5. COMMIT -> Commit com prefixo fix:
