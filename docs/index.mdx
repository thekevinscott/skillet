# Skillet

Build better Claude Code skills through systematic evaluation. Instead of guessing whether your skills work, capture real failures, measure performance, and iterate with data.

## Why Skillet?

<CardGroup cols={2}>
  <Card title="Capture Gaps" icon="crosshairs">
    Record when Claude Code fails or produces suboptimal results. Turn frustrations into test cases.
  </Card>
  <Card title="Run Evaluations" icon="chart-line">
    Systematically test your skills against captured scenarios. Get quantitative feedback on improvements.
  </Card>
  <Card title="Iterate with Data" icon="rotate">
    Make targeted improvements based on actual failure modes, not hunches.
  </Card>
  <Card title="Ship with Confidence" icon="rocket">
    Know your skills work before deploying them to production.
  </Card>
</CardGroup>

## Quick Start

Install Skillet:

```bash
pip install skillet
```

Then capture a gap, run an eval, and improve your skill.
